1. HTML

1.1 Basics


Let's start with making websites. First you need to learn HTML. This website will teach you the basics. The syntax is not too hard. You need to write your code in a <>. To colse the item use the same, but put a / in the <> BEFORE the text. Between the two items you need to put some text. Here a sample: <p>Hello world!<p> You can see here the right syntax. The <p> is a paragraph. A paragraph is a item where you can put some text in. It will apear like normal text. Check chapter 1.3 for more about text. You can add comments. It will look like: <!-- This is a comment. This is not a part of your code -->.
